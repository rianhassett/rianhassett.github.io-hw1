Object-Oriented Programming (OOP)

Object-Oriented Programming is a paradigm centered around objects and classes. A class is a blueprint for creating objects, which are instances that contain both data (attributes) and behavior (methods). OOP promotes modularity, reusability, and organized code structure.

Core Principles: Encapsulation hides internal details and protects data using access modifiers (public, private, protected). Inheritance allows classes to inherit properties from parent classes, promoting code reuse. Polymorphism enables objects to take multiple forms, allowing methods to behave differently based on context. Abstraction simplifies complex systems by exposing only necessary features.

Classes and Objects: A class defines the structure and behavior. Objects are created using the 'new' keyword. Constructors initialize objects when they're created. Methods define what objects can do, and instance variables store state.
